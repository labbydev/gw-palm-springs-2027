# Palm Springs Girls Weekend 2027

A slide deck presentation for a girls weekend trip to Palm Springs, CA — built as a single self-contained HTML file.

The presentation covers the full itinerary (Thursday–Sunday), the Aerial Tramway hike, a day at the Spa at Séc-he, dinner at Copley's, karaoke night, dive-in movie night, a budget breakdown, and why this trip works for everyone in the group.
